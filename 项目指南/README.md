## 🎯 这个项目是什么?

一句话:**多租户的 AI 智能体(Agent)SaaS 平台**,前后端分离,带完整的权限系统。

通俗讲:想象你在做一个「让企业客户都能用上 AI 助手」的产品。每个企业(租户)的数据
互相隔离;每个企业里有人是老板、有人是普通员工,能做的事不一样;AI 助手能调用工具、
能像 ChatGPT 一样流式对话。这个项目就是把这些能力打包好的**脚手架**。

👉 想快速了解全貌,先读 [`00-总览/01-项目是什么.md`](00-总览/01-项目是什么.md)。

## ✍️ 文档维护约定

为了保证这套文档长期好用,写新文档或修改时请遵守:

1. **行数限制**:每篇 ≤ 250 行(含空行)。超了就拆成 `XX-主题-续.md`。
2. **文件命名**:用 `序号-中文标题.md`,序号代表推荐阅读顺序。
3. **引用代码用符号名,不用行号**:行号会变,但「`app/repositories/base.py` 的
   `TenantScopedRepository` 类」这种锚点稳定。格式示例:
   > 多租户隔离在 `app/repositories/base.py` 的 `TenantScopedRepository` 类里实现。
4. **每篇统一结构**:开头写「这篇讲什么」,正文用小标题分段,结尾给「关键文件清单」
   和「相关文档」链接。
5. **交叉链接**:每篇顶部放「📍 相关文档」,方便跳转。
6. **语气**:对人类友好,像在跟新人面对面讲解。多用类比,少用黑话。术语首次出现时
   链接到 [术语表](附录/术语表.md)。
